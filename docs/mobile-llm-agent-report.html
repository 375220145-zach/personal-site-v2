毙掉清单 & 理由

✕
社区模板市场 (P1)
MVP 阶段用户基数不够，模板市场是双边网络效应产品——需要足够多的创作者和用户同时在线才有价值。推迟到 V1.0（Wave 2 商务用户涌入后）。
✕
分层定价 Enterprise 层 (P2)
团队管理+私有部署的 B2B 定价在 MVP 阶段不成立。先验证 C 端 PMF，B 端是企业自然上行的结果。
✕
暗黑模式 + 极简设计（作为独立方案）
融入 F1（零配置启动）和 F5（操作可视化）中，不作为独立设计项目。设计语言是执行细节，不是产品策略。
✕
日志 + 回滚机制（P0）
降级到 MVP 内置功能而非独立卖点。操作可视化（F5）天然隐含了"看到每步→可以回退"的能力。
✕
Agent 路由层自研（工程师方案）
过度工程。MVP 阶段默认走端侧推理，云端开关直接切 API，不需要复杂路由。V1.0 再加。
✕
语音唤醒（工程师方案）
息屏唤醒需要系统级权限，第三方 app 基本做不到。Android 可能用 accessibility hack 但 iOS 完全不行。移到 P2 等 OS 开放。

5. Go-to-Market 策略

定位

Pocket Agent 是为手机用户提供的零配置 AI 助手，不同于封闭的厂商 Agent（豆包/Siri）和需要部署的开源方案（AutoGLM），我们打开即用、跨 Android+iOS、数据默认不离开设备。

竞品配置平台隐私开源
Pocket Agent✅ 零配置✅ Android+iOS✅ 端侧默认✅ 核心开源
豆包✅ 零配置❌ Android定制ROM❌ 云端封闭
AutoGLM❌ 需部署❌ Android⚪ 可选✅ 开源
Siri+Gemini✅ 零配置❌ Apple生态❌ 云端封闭

定价推理链

豆包 = 免费（封闭+云端）→ Siri+Gemini = 免费（iOS内置+云端）
→ ChatGPT Plus = $20/月（通用AI+内容生成）
